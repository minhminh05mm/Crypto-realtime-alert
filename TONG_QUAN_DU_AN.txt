TONG QUAN DU AN CRYPTO REALTIME ALERT
========================================

1. Ten du an
- Crypto Realtime Alert
- Muc tieu: xay dung he thong giam sat va canh bao som bien dong gia tien ao dua tren gia real-time va tin tuc.

2. Kien truc tong the
- Du an to chuc theo monorepo.
- Gom 4 thanh phan chinh:
  + Infrastructure: cung cap Kafka, Zookeeper, Redis, ClickHouse qua Docker Compose.
  + Data Ingestion: thu thap gia va tin tuc, day vao Kafka.
  + Stream Processing: xu ly streaming bang Spark, cham sentiment bang FinBERT, sinh alert.
  + Dashboard: khu vuc de phat trien giao dien giam sat trong phase sau.

3. Luong du lieu tong quan
- Binance WebSocket -> `price_ws_client.py` -> Kafka topic `raw_prices`
- CryptoPanic API -> `news_api_client.py` -> Kafka topic `raw_news`
- Kafka `raw_prices` + `raw_news` -> `spark_pipeline.py`
- `spark_pipeline.py` -> Redis
- `spark_pipeline.py` -> ClickHouse

4. Cau truc thu muc hien tai
- `infrastructure/`
  + Chua file Docker Compose de khoi dong ha tang.
- `data_ingestion/`
  + Chua module Python de thu thap du lieu raw.
- `stream_processing/`
  + Chua Spark Structured Streaming job va NLP sentiment analyzer.
- `dashboard/`
  + Thu muc du phong cho frontend/Streamlit.
- `.env`
  + Chua toan bo bien moi truong va cau hinh van hanh.
- `Code chay file.txt`
  + Huong dan cac lenh chay du an.

5. Mo ta chi tiet tung thu muc

5.1 `infrastructure/`
- Vai tro:
  + Quan ly ha tang local cho toan bo he thong.
  + Cung cap cac service de cac module Python va Spark ket noi.

File chinh:
- `infrastructure/docker-compose.yml`
  + Khoi tao `zookeeper`
  + Khoi tao `kafka`
  + Tao san 2 topic: `raw_prices`, `raw_news`
  + Khoi tao `redis`
  + Khoi tao `clickhouse`

5.2 `data_ingestion/`
- Vai tro:
  + Dua du lieu tu nguon ngoai vao he thong.

Files:
- `data_ingestion/requirements.txt`
  + Dependency rieng cho ingestion layer.
- `data_ingestion/src/config.py`
  + Load va validate config bang pydantic-settings.
- `data_ingestion/src/kafka_utils.py`
  + Tao Kafka producer an toan, serializer JSON, retry khi startup.
- `data_ingestion/src/price_ws_client.py`
  + Ket noi Binance WebSocket.
  + Format payload gia.
  + Day gia vao topic `raw_prices`.
- `data_ingestion/src/news_api_client.py`
  + Goi CryptoPanic API theo chu ky.
  + Format payload news.
  + Day news vao topic `raw_news`.

5.3 `stream_processing/`
- Vai tro:
  + Xu ly nghiep vu streaming trung tam cua du an.

Files:
- `stream_processing/requirements.txt`
  + Dependency rieng cho Spark, Redis, ClickHouse, Transformers, Torch.
- `stream_processing/src/config.py`
  + Load config cho Spark, Redis, ClickHouse, FinBERT.
- `stream_processing/src/nlp_analyzer.py`
  + Singleton wrapper cho model FinBERT.
  + Tranh load lai model nhieu lan.
- `stream_processing/src/spark_pipeline.py`
  + Khoi tao SparkSession.
  + Doc Kafka stream.
  + Parse schema JSON.
  + Cham sentiment.
  + Join gia va news theo window.
  + Sinh alert.
  + Ghi Redis va ClickHouse.

5.4 `dashboard/`
- Vai tro:
  + Noi dat dashboard Streamlit trong phase sau.
- Hien tai:
  + Chua co code.

6. Su lien ket giua cac thu muc
- `data_ingestion/` phu thuoc vao `infrastructure/` vi can Kafka dang chay.
- `stream_processing/` phu thuoc vao:
  + Kafka de doc raw data
  + Redis de ghi hot alerts
  + ClickHouse de luu lich su
- `dashboard/` se phu thuoc vao Redis va ClickHouse de hien thi du lieu.
- `.env` la diem lien ket cau hinh chung cho toan bo he thong.

7. Schema du lieu chinh
- `raw_prices`
  + `{"symbol": "BTCUSDT", "price": 65000.50, "timestamp": 1710928492000}`
- `raw_news`
  + `{"title": "SEC approves ETF", "published_at": "2026-03-20T10:00:00Z"}`

8. Output nghiep vu
- Redis:
  + Luu alert hien tai theo symbol.
  + Phuc vu doc nhanh.
- ClickHouse:
  + Luu lich su alert theo thoi gian.
  + Phuc vu truy van, bao cao, dashboard.

9. Dinh huong mo rong
- Them Telegram Bot de gui canh bao den user.
- Them Dashboard Streamlit.
- Them quan ly schema, DLQ, monitoring, metrics, tracing.
- Them orchestration va deployment production.
